# Full Stack Testing

## Example repo of testing in the full stack dev world (FE/BE/E2E)


### How to run all of the tests?
``` bash
yarn

yarn test
```
### How to run only frontend tests?

``` bash
yarn test:app
```

### How to run only backend tests?

``` bash
yarn test:server
```

### How to run only e2e tests?

``` bash
yarn test:e2e
```
